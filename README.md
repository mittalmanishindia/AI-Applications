# AI Applications

**A growing collection of AI-enabled web applications and demonstrations.
**

This repository contains a collection of web applications and demos, including AI-enhanced tools. These are useful solutions, but they are not all full autonomous AI agents.

## What this repo contains

- `DocReader-AI-Application/`: Document reading web app for PDF, DOCX, TXT, and Markdown.
- `Outlook-Meeting-Scheduler/`: Outlook calendar scheduler using the O365 Python library.
- `PDF-Classifier-OCR/`: OCR extraction and document classification pipeline.
- `SearchEngine/`: Simple mock search engine web app built with Flask.
- `Insurance-Fraud-POC/`: Insurance fraud risk assessment with Gemini-powered reasoning (AI-assisted, not a full autonomous agent).
- `Text-Summarization-Agent/`: Text summarization API and frontend using Gemini.
- `Todo-Web-App/`: Simple client-side todo application.

## What this repo is

- A showcase of AI-enhanced utilities and examples
- A set of demos for document processing, scheduling, classification, fraud assessment, summarization, and web UI
- A starting point for building more advanced AI projects
